WORLD CUP 2026 · FIREBASE 설정

🔧 Firebase 설정 방법 (5분)

1

Firebase 접속
console.firebase.google.com 에 접속하고 구글 계정으로 로그인합니다.

2

새 프로젝트 만들기
프로젝트 추가 클릭 → 이름 입력 (예: worldcup2026) → Google 애널리틱스는 비활성화 선택 → 프로젝트 만들기

3

Realtime Database 만들기
왼쪽 메뉴에서 빌드 → Realtime Database 클릭
→ 데이터베이스 만들기 → 지역: 미국(기본값) → 테스트 모드로 시작 선택 → 완료

✅ 완료되면 화면 상단에 주소가 표시됩니다:
https://worldcup2026-xxxxx-default-rtdb.firebaseio.com
→ 이것이 databaseURL입니다. 복사해두세요.

4

웹 앱 등록 → 설정 코드 복사
왼쪽 상단 ⚙️(설정) → 프로젝트 설정 → 하단 내 앱 섹션
→ </> (웹) 아이콘 클릭 → 앱 닉네임 입력 → 앱 등록

✅ 아래와 같은 코드 블록이 나옵니다:
const firebaseConfig = {
  apiKey: "AIza...",
  databaseURL: "https://...",
  ...
};

이 코드 블록 전체를 드래그해서 복사하세요.

5

아래 칸에 붙여넣기
복사한 코드를 아래 입력창에 그대로 붙여넣고 시작하기를 누르면 자동으로 설정됩니다.
