🔧 Firebase 설정 방법 (5분)

1

Firebase 접속
console.firebase.google.com 에서 구글 계정으로 로그인

2

새 프로젝트 만들기
프로젝트 추가 → 이름 입력 → Google 애널리틱스 비활성화 → 만들기

3

Realtime Database 만들기
왼쪽 빌드 → Realtime Database → 데이터베이스 만들기 → 테스트 모드로 시작

✅ 완료 후 상단에 주소 표시 → databaseURL

4

웹 앱 등록 → 설정 코드 복사
⚙️ 프로젝트 설정 → </> 웹 아이콘 → 앱 등록

✅ const firebaseConfig = { ... } 코드 블록 전체 복사

5
